    • createHeaders

      public static HttpHeaders createHeaders(boolean isMutatorWithBody, boolean isJson)
      Creates a new HttpHeaders with some default headers filled in
      Parameters:
      isMutatorWithBody - if true, sets the content type. If isJson is false, sets to application/x-www-form-urlencoded
      isJson - if true, sets the accept to application/json. If isMutator is true, sets the content type as well
      Returns:
      a HttpHeaders with some default headers, if the parameters require them
